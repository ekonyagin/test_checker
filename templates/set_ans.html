Создать новый тест

Тема теста - это его уникальное имя (на русском языке). Ответ на вопрос - это последовательность номеров (ЦИФР) правильных ответов БЕЗ разделителей! Номера можно писать в любом порядке. Любые другие символы будут проигнорированы.

При совпадении имени теста + номера варианта с уже имеющимся будет сохранена последняя версия!

Просмотреть темы и варианты уже записанных тестов можно здесь.
